Qiskit Surface Code Encoder and MWPM Decoder

Open-source rotated surface code package for Qiskit Ignis created by YuQC members.

Nov 11, 2020

About

We are building a Surface Code Encoder and Minimum Weight Perfect Matching Decoder package for Qiskit Ignis.

Here is the Github repo: https://github.com/Phionx/qiskit_surface_codes

Here is a link to our project proposal.

Status

This open-source package was presented at the November 2020 Qiskit Advocate Demo Session and is pending official incorporating into Qiskit Ignis. Our immediate goal is to demonstrate the rotated surface code threshold point in simulation.

Team

The first iteration of this project was created by members Shantanu Jha ‘21, Henry Liu ‘21 and Will Sun ‘22 for IBM Qiskit Ignis. The team has grown since then, and a full roster will be posted soon!

Contact

If you’re interested in joining this project, please either sign up on this page or email shantanu.rajesh.jha [at] gmail.com.
